


# CyberEden (nazrOS)

Цифровая киберсистема от nazrOS — платформа для кибержурнала, маркетплейса, хакатонов, стримов и конструктора ОС.

## 🚀 Стек

- **Фронтенд:** React 19 + TypeScript
- **Сборщик:** Vite 5
- **Маршрутизация:** TanStack Router (SPA)
- **Стили:** Tailwind CSS 3 + PostCSS
- **Анимации:** Framer Motion
- **UI-компоненты:** Lucide React
- **Хостинг / CDN:** Cloudflare Pages (Edge Network)

## 🛠️ Установка

```bash
git clone https://github.com/nazrOS-softhard/Gitverse-cybereden_v1.git
cd Gitverse-cybereden_v1
npm install
npm run dev
```

## 📦 Сборка

```bash
npm run build
```

## 🌐 Деплой

Проект настроен для деплоя на **Cloudflare Pages**:

1. Открой Cloudflare Pages → Connect to Git → выбери репозиторий.
2. Настрой сборку:
   - Build command: `npm run build`
   - Output directory: `dist`
3. Нажми Deploy.

## 🎮 Игровая механика

### СИСТЕМА РАНГОВ nazrOS // CYBEREDEN

Ты строишь не просто систему ролей, а полноценную MMORPG-механику цифровой операционной среды. Главная идея: пользователь не «получает роль», а развивается внутри киберсистемы.

#### 4 КЛАССА УЧАСТНИКОВ

| Класс | Роль | Описание |
|-------|------|-----------|
| **НАБЛЮДАТЕЛИ** | Пользователи среды | Читают журнал, участвуют в стримах, тестируют интерфейсы |
| **ОПЕРАТОРЫ** | Активные участники | Работают с репозиториями, создают HUD, участвуют в хакатонах |
| **АРХИТЕКТОРЫ ЯДРА** | Создатели технологий | Разрабатывают софт, устройства, проектируют модули |
| **ГЛАВНЫЕ РАЗРАБОТЧИКИ** | Презентуют устройства, ведут направления, принимают решения |

#### РАНГИ

| Символ | Название | Описание |
|--------|----------|-----------|
| ○ | Без оценки |  Активный участник |
| △ | Треугольник | Инженерный статус (выдаётся вручную) |
| □ | Квадрат | Ядро системы (для архитекторов и главных разработчиков) |


#### УРОВНИ

Каждый ранг имеет уровни от 1 до 9.  
Пример: `Наблюдатель ○ 1`, `Оператор □ 7`, `Архитектор △ 3`

#### XP СИСТЕМА

**Типы XP:**

- **MEDIA XP** — статьи, стримы, публикации, видео, визуал, репортажи, загруженные ассеты
- **ENGINEERING XP** — коммиты, merge, pull request, исправления, архитектура, backend/frontend
- **SYSTEM XP** — участие в жизни среды, тестирование, баг-репорты, модерация, поддержка
- **EVENT XP** — турниры, хакатоны, доклады, презентации, участие в событиях

**Примеры начисления:**

| Действие | XP |
|----------|----|
| Статья | +150 XP |
| Стрим | +300 XP |
| Pull Request | +250 XP |
| Merge accepted | +500 XP |
| Найден критический баг | +700 XP |
| Победа в хакатоне | +2500 XP |
| Презентация устройства | +4000 XP |

#### АНТИСПАМ МЕХАНИКА

**QUALITY SCORE** — каждый материал оценивается по активности, реакциям, времени просмотра, ревью, инженерной оценке.

**Мультипликаторы:**
- NIGHT SESSION (ночные стримы) — ×1.2 XP
- CORE EVENT (официальные ивенты) — ×2 XP
- SYSTEM CRITICAL (критические ошибки) — ×3 XP

#### СПЕЦИАЛИЗАЦИИ

У каждого пользователя может быть специализация: HUD Engineer, Motion Architect, Core Researcher, Signal Operator, Cyber Athlete, System Builder, Protocol Designer.

#### РЕПУТАЦИЯ

Reputation Score — доверие системы, которое дают архитекторы, комьюнити, модераторы, лидеры департаментов.

#### ДОСТУП К СРЕДЕ

Чем выше ранг, тем больше HUD, модулей, кастомизации и возможностей.

#### СЕЗОНЫ

- `SEASON_01 // INITIALIZATION`
- `SEASON_02 // CORE EXPANSION`
- `SEASON_03 // DIGITAL ASCENSION`

В конце сезона игрок получает редкие статусы, уникальные цвета HUD, цифровые предметы, анимированные эмблемы, архивные достижения.

#### ИВЕНТЫ

- **BUILD EVENT** — создание интерфейсов
- **SIGNAL EVENT** — медиа-события
- **GAMMA TEST** — тестирование систем
- **CORE WAR** — командные инженерные соревнования
- **BLACK PROTOCOL** — секретные ARG-события
